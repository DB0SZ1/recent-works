# Recent Works

A high-performance collection of polished UI components, interactive web apps, and modern landing pages built with Next.js and TypeScript. 

## Tech Stack
- **Framework:** Next.js 16 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS v4
- **Animation:** Framer Motion, `@paper-design/shaders-react`, `tw-animate-css`
- **UI Components:** Radix UI, shadcn/ui
- **Icons:** Lucide React

## Project Overview

This repository contains several complex UI implementations and landing pages designed with a focus on modern web aesthetics, dynamic animations, and optimal user experience.

### Pages & Components Highlight

- **E-commerce Landing (`components/landings/ecommerce-landing.tsx`)**
  A robust and fully responsive e-commerce storefront landing page featuring dynamic product grids, smooth transitions, and a modern aesthetic tailored for online retailers.

- **AI Pipeline Landing (`components/landings/ai-pipeline-landing.tsx`)**
  A tech-forward landing page showcasing artificial intelligence product features. It includes sleek data visualization components, modern typography, and a dark-mode optimized layout.

- **Crypto Landing (`components/landings/crypto-landing.tsx`)**
  A premium, high-trust cryptocurrency platform landing page with subtle micro-animations, glassmorphism effects, and engaging call-to-actions.

- **Geometric Hero (`components/mvpblocks/geometric-hero.tsx`)**
  A visually striking hero section component that utilizes complex geometric patterns and dynamic interactions to immediately capture user attention.

- **Playground (`app/playground/page.tsx`)**
  An interactive testing environment used to rapidly prototype, iterate, and preview UI components in isolation.

- **Hero Dithering Card (`components/ui/hero-dithering-card.tsx`)**
  An advanced hero section that leverages WebGL shaders (via `@paper-design/shaders-react`) to create a mesmerizing fullscreen dithering effect. It includes sophisticated typography and hover-responsive shader animation speeds.

## Delivery Timeline

- **Core Pages & Components:** Delivered in a rapid turnaround time of **16 hours**. This includes the E-commerce, AI Pipeline, and Crypto landing pages, as well as the Geometric Hero and Playground setups.
- **Hero Dithering Card (`hero-dithering-card.tsx`):** Delivered in **28 hours**. The extended timeline was solely due to pending client feedback and response delays regarding the specific shader interactions and visual tuning.

## Getting Started

First, install the dependencies:

```bash
npm install
# or
yarn install
# or
pnpm install
```

Then, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
